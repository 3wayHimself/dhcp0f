dhcspy
======

Passive DHCP analyzer with OS fingerprinting on the LAN through DHCP


Requirements
------------

* Linux (work in progress for other OS)
* perl 5.8+
* Net::Pcap


Install
-------

**Instructions are incomplete**

1. `git clone ...`
2. install Net::Pcap
3. ./dhcspy.pl -i <interface>
